# 🎭 Sistema de Reacciones Contextuales para Finanzas Familiares

## 📖 Visión General

El sistema de reacciones permite a las familias comunicarse de forma más expresiva y contextual sobre sus transacciones financieras. **12 emojis cuidadosamente seleccionados** cubren las reacciones más comunes en contexto de finanzas familiares.

### 🎯 Principios de Diseño (v1.5 - Facebook-Style)
- ✅ **Menos es más:** 12 emojis esenciales vs 38 abrumadores
- ✅ **Centrado:** Modal fijo que no se sale del viewport
- ✅ **Touch-friendly:** Grid 3x4 optimizado para móvil
- ✅ **Overlay oscuro:** Contexto visual y fácil de cerrar
- ✅ **Labels visibles:** Excepto durante animación de selección
- ✅ **3 categorías:** Positivo, Negativo, Neutro
- ✅ **Accesible:** Cierra con overlay, ESC, o después de seleccionar
- ✅ **Feedback háptico:** Vibración 50ms al seleccionar (estilo Facebook)
- ✅ **Animación simultánea:** Emoji seleccionado salta, otros desaparecen
- ✅ **Rápido:** 250ms de animación + cierre inmediato

## 🎯 **Casos de Uso Principales**

### 1. **Validación Social**
Los miembros pueden expresar aprobación/desaprobación sin conflicto directo:
- ✅💚👍 = aprobado por la familia
- 😰😬🚨 = señales de alerta

### 2. **Educación Financiera**
Los stickers ayudan a crear conciencia:
- 💰🎯🧠 = decisiones inteligentes que se replican
- 💸😱🤯 = gastos que generan reflexión

### 3. **Construcción de Cultura**
Cada familia desarrolla su propio "lenguaje":
- Familia A: usan 😂🤣 para gastos innecesarios (con humor)
- Familia B: usan 🚨⚠️ para gastos que exceden presupuesto
- Familia C: usan 🎉🥳 para celebrar ahorros logrados

### 4. **Reducción de Conflictos**
Emojis suavizan críticas directas:
- En vez de: "¿Por qué compraste eso?"
- Usan: 🤔👀 (pregunta sin confrontación)

## 🛠️ **Implementación Técnica**

### Estructura de Datos
```typescript
interface EmojiReaction {
  emoji: string;        // El emoji Unicode
  label: string;        // Descripción corta
  category: string;     // Categoría para agrupación
}
```

### Categorías Disponibles
1. `aprobacion` - Positivo, aprobado
2. `preocupacion` - Negativo, alerta
3. `inteligente` - Decisión sabia
4. `sorpresa` - Shock, caro
5. `humor` - Gracioso, diversión
6. `duda` - Preguntas, sospecha
7. `celebracion` - Logro, festejo
8. `neutral` - Sin opinión
9. `impresionante` - Wow, increíble
10. `empatia` - Apoyo emocional
11. `apoyo` - Solidaridad
12. `alerta` - Cuidado, alto

### UI/UX
- **Picker:** Grid 8x5 (38 emojis + espacio)
- **Tooltips:** Label descriptivo al hacer hover
- **Destacado:** Borde verde esmeralda si el usuario ya reaccionó
- **Contador:** Muestra cuántas personas reaccionaron con ese emoji
