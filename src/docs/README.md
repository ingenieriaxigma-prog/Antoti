## 🎯 Bienvenido a Oti

**Oti** es tu asistente personal de finanzas inteligente que te ayuda a:

✅ Registrar ingresos y gastos fácilmente  
✅ Controlar presupuestos mensuales  
✅ Visualizar estadísticas financieras  
✅ Usar reconocimiento de voz para transacciones rápidas  
✅ Chatear con un asistente IA para consejos financieros  
✅ Mantener tus datos seguros con encriptación

## 🚀 Inicio Rápido

### **Para Usuarios**

1. **Abre Oti** en tu navegador
2. **Crea tu cuenta** (email + contraseña)
3. **Completa el tour** interactivo
4. **Crea tus cuentas** (efectivo, banco, etc.)
5. **Registra tu primera transacción** 💸

✨ **¡Ya estás listo!**

**¿Necesitas ayuda?** → [Guía de Usuario](/docs/USER_GUIDE.md)

---

### **Para Desarrolladores**

```bash
# 1. Clonar repositorio
git clone https://github.com/tu-usuario/oti.git
cd oti

# 2. Instalar dependencias
npm install

# 3. Configurar .env
cp .env.example .env
# Editar .env con tus keys de Supabase

# 4. Ejecutar en desarrollo
npm run dev

# 5. Abrir http://localhost:5173
```

**¿Necesitas ayuda?** → [Guía de Desarrolladores](/docs/DEVELOPER_GUIDE.md)
